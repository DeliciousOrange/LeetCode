# 思路（C#）
这是LeetCode上遇到的第一道题目，由于之前没有学过数据结构的知识，所以每次遇到这种需要遍历数组元素的题目，总是选择使用暴力解法。这次，在暴力解法之外考虑用更优的解法来解决这个问题。

在使用c#的dictionary解决问题的时候，要注意不像java中的hashmap，dictionary的key一定不能重复，所以Solution2在往dictionary中添加键值对时需要判断是否已经存在该键。
